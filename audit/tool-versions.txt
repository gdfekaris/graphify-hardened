Trust-scope note for osv-scanner install:
  The SHA256 above was verified against osv-scanner_SHA256SUMS published in the
  same v2.3.5 GitHub release. This protects against bit-rot and CDN tampering
  but not against compromise of the GitHub release itself. Stronger
  verification would use `cosign verify-blob` against the sigstore signature
  osv-scanner publishes. For a private hardened fork run locally this level
  is accepted; revisit if/when this fork goes public.
