vis-network vendoring decision and pin
========================================

Recorded: 2026-05-01
Plan tasks: 2.1 (decision), 2.2 (version + hash)

Task 2.1 — Vendoring strategy
-----------------------------
Decision: INLINE (vendor the JS into graphify/static/, embed it in every
generated graph.html via importlib.resources). Selected for the strongest
security posture: self-contained output, works offline, no third-party
runtime trust, no CDN-tampering risk.

SRI-with-CDN was considered and rejected: a tampered CDN response would
fail closed (browser blocks the script) but would also break the graph
viewer; inline avoids both the trust dependency and the failure mode.
